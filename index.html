TABLA DE CONTENIDO

Consolidado 1-Parte I

Semana 1: Tecnologias de Desarrollo Web

Definiciones

Las tecnologías de desarrollo web se refieren a un conjunto de herramientas, lenguajes de programación, frameworks y estándares utilizados para crear aplicaciones y sitios web en Internet.

Procedimientos

Algunos componentes de las tecnologias de desarrollo web se dviden en:

Lado del cliente Frotend; cuyos lenguajes de programación principales son HTML, CSS y Java Script

Lado del cliente backend; tenemos servidores web y lenguajes de programación entre los conocidos PHP, Java, Python entre otros

Resultados

Se aprendio a diferenciar las diferentes tecnologias web que hay y que permiten a los desarrolladores crear aplicaciones web efectivas y funcionales, desde la estructura y el diseño hasta la funcionalidad y la gestión de datos.

Semana 2: HTML

Definiciones

HTML es un lenguaje estandar que se utiliza para definir la estructura y el formato de una página web, lo que permite la presentación de texto, imágenes, enlaces, formularios y otros elementos multimedia en un navegador web.

Procedimientos

Dentro de las caracteristicas que tiene y ofrece html estan:

Nos permite organizar el contenido de una página en encabezados, párrafos, listas, tablas, formularios y otros elementos para facilitar la comprensión y la navegación.

Tambien nos permite crear hipervinculos o enlaces (links) ademas de crear formularios que diferentes usuarios pueden visitar o rellenar con datos o comentarios.

Resultados

Se aprendio que HTML tiene diferentes funciones como crear páginas web y definir la estructura y contenido de una página, ademas de que trabaja de la mano con CSS y Java Script

Semana 3: CSS y Liberias

Definiciones

CSS es un lenguaje utilizado para controlar la presentación y el diseño de páginas web, que se se centra en aspectos visuales como colores, fuentes, márgenes, tamaños y posiciones de elementos en la página.

Procedimientos

Dentro de sus funciones CSS nos permite tener un diseño coherente que facilita la creación de diseños consistentes en un sitio web.

Tambien trabaja con Liberias como: Bootstrap (Un framework muy utilizado que proporciona una colección de estilos predefinidos y componentes reutilizables para la creación rápida de sitios web responsivos.) y Less que añaden funcionalidades avanzadas como variables, anidamiento y mixins para simplificar la escritura de código

Resultados

En general CSS nos permite separar el contenido de la presentación, lo que facilita la creación de diseños consistentes y atractivos en un sitio web ayudandose de muchas librerías y frameworks CSS que facilitan la creación de diseños y componentes web avanzados.

Consolidado 1-Parte II

Semana 4: Java Script

Definiciones

JavaScript es un lenguaje de programación de alto nivel, interpretado por los navegadores web, que se utiliza para hacer que las páginas web sean interactivas y dinámicas.

Procedimientos

JavaScript tiene múltiples usos y funcionalidades en el desarrollo web como:

Interactividad en el lado del cliente, Comunicación con el servidor, ya que permite que enviar y recibir datos del servidor de manera asincrónica a través de tecnologías como XMLHttpRequest y la API Fetch y una automatización en el navegador web

Resultados

JavaScript es un lenguaje de programación esencial para el desarrollo web, ya que agrega interactividad y dinamismo a las páginas web, lo que mejora la experiencia del usuario y permite la creación de aplicaciones web avanzadas.

Semana 5: Desarrollo con React

Definiciones

React es una libreria de JavaScript que sirve para crear interfaces de usuario, tambien se utiliza para crear SPA aplicaciones de una sola pagina

Procedimientos

React se utiliza para una variedad de propósitos en el desarrollo web y móvil como:

Desarrollo de aplicaciones web interactivas ya que permite crear interfaces de usuario dinámicas y altamente interactivas para aplicaciones web, tambien el desarrollo de aplicaciones móviles utilizando React Native y una integración con otras bibliotecas y frameworks

Resultados

En resumen, React es una biblioteca de JavaScript utilizada para construir interfaces de usuario dinámicas y eficientes en aplicaciones web y móviles por su rendimiento optimizado y su capacidad de desarrollo rápido.
